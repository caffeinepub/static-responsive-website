# Static Responsive Website

## Overview
A static responsive website with a hero section, image carousel, and basic navigation structure featuring a comprehensive multi-level dropdown menu system.

## Features

### Hero Section
- Large background image or video display
- Main title and subtitle text overlay
- Call-to-action button
- Full-width responsive layout

### Image Carousel
- Slider/carousel component below hero section
- Displays multiple images or banners
- Navigation controls (previous/next buttons)
- Auto-play functionality
- Touch/swipe support for mobile devices

### Navigation
- Header with navigation menu
- Basic links (Home, About, Services, Contact)
- **Multi-level dropdown menu under "SERVICES" with the following structure:**
  - **PAINTINGS**: Realistic, Portrait, Abstract, Wall Art (2D & 3D)
  - **ARTWORKS**: Murals, Nameplates, Gift Articles, Interior Artwork
  - **SIGNAGES**: Branding Signs, LED Signs, Informative Signs, Building Names
  - **SCULPTURES**: Mural & Statue, Fiber, Metal, Stone
  - **PRINT SOLUTION**: UV Printing, Eco Solvent, Transparent, One-way Vision, Customize Wallpapers
  - **GRAPHICS (Design)**: Logo Design, Visiting Card, Company Profile Design, Brochure Design, ID Card Design, Packaging Design, Flyers Design, Poster Design, Menu Card Design, Exhibition Stall Design, Invitation Design, Festival Greeting Design
  - **GRAPHICS (Digital)**: UI/UX, Website Design, Social Media Post Design
- Dropdown behavior: hover activation on desktop, tap toggle on mobile
- Mobile-responsive hamburger menu with collapsible dropdown sections
- Footer with additional links and basic information

### Responsive Design
- Mobile-first approach
- Breakpoints for mobile (320px+), tablet (768px+), and desktop (1024px+)
- Flexible grid layout
- Optimized images and content for different screen sizes
- Dropdown menus adapt to screen size with appropriate spacing and touch targets

## Content
- All content and interface elements in English
- Placeholder content for demonstration purposes
- Sample navigation links and footer information
- Service categories and subcategories as specified

## Technical Requirements
- Static website (no backend data storage required)
- Client-side only functionality
- Modern web standards compliance
- Cross-browser compatibility
- Smooth dropdown animations and transitions
